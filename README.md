# tiny_transformer
一个简单的例子使用transformer模型进行英译中任务
# 训练
```
  python model.py
```
# 测试结果
```text
He is my youngest brother . EOS
BOS 他 是 我 最 年 轻 的 兄 弟 。 EOS

BOS Most boys like computer games . EOS
BOS 大 多 数 男 生 喜 欢 电 脑 游 戏 。 EOS
translation: 大 部 分 男 生 喜 欢 电 脑 游 戏 。
```
